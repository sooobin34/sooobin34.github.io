## 🐶 프로젝트 개요
애견인을 위한 **분양 정보, 지역 기반 애견카페/병원, 용품 거래 게시판** 등을 제공하는 웹사이트를 개발했습니다.

## ⚙️ 주요 기능
- 회원가입 및 로그인  
- 용품 직거래 게시판  
- 지역별 병원 및 카페 정보 제공  
- 이미지 업로드 및 댓글 시스템  

## 🧩 기술 스택
- **Backend:** Flask (Python)  
- **Database:** Oracle SQL Developer  
- **Frontend:** HTML, CSS, JavaScript  

## 💡 배운 점
실제 데이터베이스를 설계하고 웹과 연동하면서  
**ERD 설계 → SQL 구현 → 웹 연동**의 전체 흐름을 체계적으로 경험했습니다.
