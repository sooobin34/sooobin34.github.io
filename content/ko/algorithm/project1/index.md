## 📘 프로젝트 개요
알고리즘의 동작 과정을 쉽게 이해할 수 있도록 **정렬, 탐색, 그래프 알고리즘**을 시각화하는 툴을 만들었습니다.

## 🧩 주요 기능
- 선택 정렬, 버블 정렬, 퀵 정렬 등 다양한 알고리즘 지원  
- 단계별 실행, 속도 조절 기능  
- Canvas를 이용한 시각화  

## ⚙️ 사용 기술
- **Frontend:** HTML, CSS, JavaScript  
- **Library:** D3.js  

## 💡 배운 점
복잡한 알고리즘 로직을 시각적으로 표현하면서 **시간 복잡도와 데이터 구조의 중요성**을 깊게 이해하게 되었습니다.
